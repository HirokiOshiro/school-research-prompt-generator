## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | School Research Prompt Generator |
| **目的** | 留学生入試担当職員が、志願者の出身高校情報を調査するための生成AI用プロンプトを簡単に生成できるWebツール |
| **対象ユーザー** | 入試担当職員（日本語話者・英語話者両方） |
| **利用環境** | Windows PC、ブラウザ |
| **生成AI利用先** | Microsoft Copilot（社内契約）、その他個人判断のAI |
| **ホスティング** | GitHub Pages |
| **本番URL** | https://hirokioshiro.github.io/school-research-prompt-generator/ |
| **リポジトリ** | https://github.com/HirokiOshiro/school-research-prompt-generator |
| **技術スタック** | HTML + CSS + Vanilla JavaScript（フレームワーク不使用） |

## 2. 機能要件

### 2.1 入力機能

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|---------|------|
| 高校名 | 必須 | テキスト入力 | 英語・現地語どちらでも可 |
| 所在国 | 必須 | ドロップダウン + その他入力 | 10カ国 + 「その他」選択時にテキスト入力 |
| 都市/州 | 任意 | テキスト入力 | |
| 学校種別 | 任意 | ドロップダウン | 公立/私立/インターナショナル/不明 |
| カリキュラム | 任意 | チェックボックス複数選択 | IB, A-Level, AP, 国内課程, 不明 |
| 調査目的 | 必須 | ラジオボタン | 新規登録/情報更新/資格確認 |
| 現地語検索指示を含める | 任意 | チェックボックス | デフォルトOFF |
| 追加参照URL | 任意 | テキスト入力 | カンマ区切りで複数入力可 |

### 2.2 出力機能

| 機能 | 説明 |
|------|------|
| プロンプト生成 | 入力内容に基づき調査用プロンプトを1つ生成 |
| コピーボタン | ワンクリックでクリップボードにコピー |
| リセットボタン | 入力内容をクリア |

### 2.3 国別プロンプト強化

選択された国に応じて、既存エージェント定義の以下情報をプロンプトに自動挿入：

- 教育制度の概要（学制・年数）
- 学校種別の分類（Academic vs Vocational）
- 公式データソースURL
- 現地語での検索クエリ例
- 注意事項（職業学校の判別等）
